    Méthode display

    La méthode display choisit le shader et la texture appropriée via la méthode bind sélectionne le tableau de vertices et d’indices et fait l’appel de rendu.

      public display() {
    GL.bindBuffer(GL.ARRAY_BUFFER, this.vertexBuffer);
    GL.bindBuffer(GL.ELEMENT_ARRAY_BUFFER, this.indexBuffer);
    this.bind();
    GL.drawElements(GL.TRIANGLES, 6, GL.UNSIGNED_SHORT, 0);
    this.unbind();
  }
